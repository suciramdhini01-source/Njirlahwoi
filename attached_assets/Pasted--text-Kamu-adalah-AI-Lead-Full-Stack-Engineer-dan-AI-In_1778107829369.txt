## 🌐 1. NJIRLAH AI – KONSEP UTAMA
NJIRLAH AI adalah platform chat AI multi‑model dengan dua mode:
1. **Mode Tamu (Guest)** – Pengguna **TIDAK PERLU API key**. Aplikasi menggunakan API key gratis bawaan (dari environment variable server) untuk mengakses **seluruh model gratis OpenRouter** yang terdaftar di bawah.
2. **Mode BYOK (Bring Your Own Key)** – Pengguna memasukkan kunci OpenRouter dan/atau Cloudflare Workers AI miliknya sendiri. Kunci **tidak pernah disimpan di server**, hanya dienkripsi di localStorage. Dengan BYOK, pengguna dapat mengakses **semua model dari 55 provider** yang didukung OpenRouter.

Branding: **edgy, playful, premium** – kombinasi Tailwind glassmorphism + animasi WebGL dari Unicorn.Studio + warna neon purple/cyan/pink. Nama situs: **NJIRLAH AI**. Tagline: “Chat AI Tersesat, Bebas Pake Kunci Sendiri”.

## 🔐 3. KEAMANAN & ARSITEKTUR BYOK (THE LAW)
1. Semua kunci pengguna **hanya disimpan di localStorage**, dienkripsi dengan AES‑GCM (Web Crypto API). Passphrase turunan dari kombinasi unik browser.
2. API routes server hanya bertindak sebagai **proxy transparan**:
   - `POST /api/openrouter/chat` – membaca header `x-api-key` dari request, meneruskannya ke OpenRouter **tanpa menyimpan**.
   - `POST /api/cloudflare/chat` – membaca `x-cf-token` dan `x-cf-account-id`, meneruskan ke Cloudflare.
3. **Tidak boleh** menyimpan kunci di database atau environment variable server, **KECUALI** satu API key gratis untuk mode tamu (lihat bagian 4).
