## 1. DESKRIPSI APLIKASI
NJIRLAH AI adalah platform chat multi-model AI yang mendukung lebih dari 300+ LLM melalui integrasi langsung dengan penyedia utama: **OpenRouter**, **Cloudflare Workers AI**, **Alibaba Cloud Bailian**, **Cline (OpenAI-Compatible)**, dan **seluruh penyedia BYOK OpenRouter**. 

Aplikasi ini memiliki halaman utama **Chat** (`/`) dan halaman penting baru bernama **"API NJIR"** (`/api-njir`), yang menjadi pusat kontrol untuk mengelola, mengonfigurasi, dan menguji semua API key penyedia.

---

## 2. TECH STACK (WAJIB)
- **Next.js 15** (App Router) + TypeScript strict
- **Tailwind CSS 4** + custom theme (neon purple/cyan/pink, glassmorphism, dark mode)
- **shadcn/ui** untuk komponen dasar
- **Framer Motion**, **GSAP**, **react-spring**, **anime.js** (untuk animasi kaya dan UI premium)
- **Zustand** untuk state management API key dan model
- **Web Crypto API** untuk enkripsi semua API key di localStorage
- **Vercel** untuk deployment

### 3.2 Cloudflare Workers AI (Langsung)
- Pengguna memasukkan **Cloudflare Account ID** dan **Cloudflare API Token**.
- Cara mendapatkannya: [Panduan Resmi Cloudflare](https://developers.cloudflare.com/workers-ai/get-started/rest-api/). Token didapat dari dashboard Cloudflare > Workers AI > Use REST API > Create API Token.
- Model yang tersedia: @cf/meta/llama-3.1-8b, @cf/mistral/mistral-7b, @cf/deepseek/deepseek-r1, dan lain-lain (di-fetch otomatis dari API).

### 3.3 Alibaba Cloud Bailian (Langsung)
- Pengguna memasukkan **Bailian API Key**.
- Cara mendapatkannya: [Panduan Resmi Alibaba Cloud](https://developer.aliyun.com/article/1697678). Masuk ke [Konsol Bailian](https://bailian.console.aliyun.com/), buka API Key Center, buat kunci baru.
- Model utama: Qwen-Turbo, Qwen-Plus, Qwen-Max, Llama-3, DeepSeek-V3/R1, dll.

### 3.4 Cline (OpenAI-Compatible Provider)
- Pengguna bisa menambahkan endpoint pihak ketiga yang kompatibel. Masukkan:
  - **Nama Provider**: Bebas (contoh: "Cline Local", "SiliconFlow", "RouterPark", dll.)
  - **Base URL**: Endpoint yang kompatibel dengan API OpenAI, contoh:
    - `https://api.siliconflow.cn/v1`
    - `http://localhost:11434/v1` (untuk Ollama lokal)
  - **API Key**: Kunci autentikasinya.
  - **Model ID**: Nama model spesifik (contoh: `meta/llama-3.1-8b-instruct`)
- Ini memungkinkan integrasi dengan 100+ model dari 20+ penyedia tambahan yang tidak ada di daftar utama.

## 4. HALAMAN "API NJIR" (`/api-njir`)
Ini adalah halaman kustom yang WAJIB dibangun sebagai pusat pengelolaan kunci API. Konsepnya adalah "Dashboard Untuk Semua Kunci Anda".

### 4.1 Desain & Layout
- **Header**: Judul "API NJIR" dengan sub-judul "Kendalikan Semua Kunci API di Satu Tempat".
- **Tata Letak**: Gunakan sistem grid atau tab.
  - **Tab 1: OpenRouter** -> Input API Key utama + sub-grid untuk 56 BYOK providers.
  - **Tab 2: Cloudflare** -> Input Account ID & Token.
  - **Tab 3: Alibaba Bailian** -> Input API Key.
  - **Tab 4: Custom (Cline)** -> Form dinamis untuk menambah endpoint khusus.

### 4.2 Komponen Fungsional Wajib
- **Input Field**: Desain glassmorphism, dengan ikon gembok. Ada tombol "Test Connection" yang akan mem-ping endpoint sederhana untuk validasi.
- **Status Indicator**: Lampu hijau/merah untuk menunjukkan apakah kunci valid atau tidak.
- **Encrypt & Save**: Setiap kali kunci disimpan, enkripsi dan simpan ke localStorage.
- **Daftar Model**: Di bawah setiap input penyedia, tampilkan badge model-model yang tersedia. Untuk OpenRouter, fetch dan tampilkan 300+ model yang bisa diakses.

### 4.3 Fitur "Tambah Custom Provider"
- Pengguna bisa menekan tombol "Add Custom Provider".
- Akan muncul modal/form untuk mengisi: Nama Provider, API Base URL, API Key, dan Model ID.
- Data ini disimpan terenkripsi dan akan muncul di daftar penyedia yang bisa dipilih saat chatting.
